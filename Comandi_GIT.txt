- git intit  ➡️ Per creare una Repository Locale; 

- git status ➡️ Effettua un controllo sullo stato della Repository; 

- git add Nome_file ➡️ traccia i file nuovi o modificati; 

- git add .ext ➡️ traccia tutti i file con quella estensione (.ext); 

- git add . ➡️ traccia tutti i file; 

- git rm --cached nomeFile ➡️ rimuove i file appenna aggiunti nella Stagin Area (con add) per non salvarli nella repository locale; 

- git commit -m "messaggio del commit" ➡️ salvo i file tracciati nella Stagin Area nella Repository locale; 

- git branch -M main ➡️ colleghiamo il branch principale di GIT (main) con il branch principale di GitHub (Master); 

- git remote add origin https://github.com/FeliCarlomagno/GIT.git ➡️ collego il repository locale a quello remoto; 

- git push -u origin main ➡️ va a salvare in remoto tutto ciò che avevo in locale sul branch di origine mian; 

- git branch nomeBranch ➡️ crea una ramificazione con il nome indicato;

- git checkout nomeBranch ➡️ sposta l'area di lavoro su un branch ramificato o per tornare sul branch principale; 

- git branch ➡️ stampa un elenco di branch presenti nel progetto;
